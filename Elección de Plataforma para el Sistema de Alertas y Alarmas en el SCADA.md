# Título
Elección de Plataforma para el Sistema de Alertas y Alarmas en el SCADA

## Estado
Propuesto

## Contexto
El cliente, responsable de la gestión de la red eléctrica de alta tensión en tiempo real (Transmission System Operador, TSO), necesita seleccionar la plataforma tecnológica para la gestión de alertas y alarmas en el nuevo sistema SCADA.

## Decisión
Se optará por implementar un sistema de gestión de alertas y alarmas basado en una plataforma de código abierto.

Se utilizará un mecanismo de notificación en tiempo real para alertar al operador sobre incidencias críticas.

## Consecuencias
Se requerirá la asignación de recursos para la implementación y personalización de la plataforma de código abierto.

La notificación en tiempo real demandará una infraestructura robusta y conexiones de alta velocidad.
